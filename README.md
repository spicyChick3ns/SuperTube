# SuperTube
SuperTube is a Youtube derivative that uses React and Youtube APIs.

### Getting Started:
A youtube API key is needed to run SuperTube. Included with a live search feature.
```
git clone git@github.com:spicyChick3ns/SuperTube.git
npm install
npm start
```
